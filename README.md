# forum-vue
基于 vue + vuex + element-ui + markdown-it

使用markdown编辑文章

服务端地址 https://github.com/blowsnowit/forum-server

## 演示地址
http://forum.bload.cn/

## 已实现的功能
- 发布文章
- 评论功能
- 话题功能
- 登录，注册，找回，修改密码等
- 图片上传
- 消息功能
  - xxx回复了你（点击跳转到对应的评论区）
  - xxx当前登录时间ip 时间
  - 公告
- 后台
- 标签功能
- 文章 全文索引
